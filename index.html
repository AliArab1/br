Examples of what it does

Printer stuck?

Clear the print queue, restart the spooler, re-test.

Revoka generates a PowerShell script that stops the print spooler service, clears all pending print jobs from the queue, restarts the service, and verifies the printer is responding. All actions are logged and can be undone within 60 seconds if needed.
Wi-Fi broken?

Reset the adapter and network settings, reconnect.

Automatically disables and re-enables the Wi-Fi adapter, flushes DNS cache, resets TCP/IP stack, removes saved network profiles if corrupted, and re-establishes connection. The script includes rollback to restore previous network configurations.
Mic/webcam dead?

Reset permissions, rebind the device, quick test.

Checks device permissions in Windows settings, resets camera and microphone privacy settings, uninstalls and reinstalls device drivers, and runs a quick hardware test to verify functionality. All permission changes are tracked for audit.
VPN/DNS failing?

Flush DNS, repair split-tunnel, reachability check.

Flushes DNS resolver cache, resets network adapter configurations, repairs VPN split-tunnel settings, tests connectivity to VPN endpoints, and verifies DNS resolution. Includes automatic rollback if connectivity worsens.
Fake login page detected?

Close the tab, lock the session, revoke tokens.

Immediately closes the phishing tab, locks the Windows session to prevent further access, revokes all active Google and Microsoft 365 authentication tokens, logs the incident with timestamp and user details, and provides a 60-second undo window in case of false positive.
Outlook not syncing?

Reset Office credentials, clear cache, re-authenticate.

Clears Outlook cached credentials, removes stored authentication tokens, resets Office 365 connection settings, clears local mailbox cache, and prompts for fresh authentication. All credential operations are logged for security compliance.
